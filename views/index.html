Image Search microservice

How does this work?

  1. Call the /api/imagesearch endpoint followed by the name of the term you wish to search for to get image data.

  2. Add an ?offset= followed by an integer to get a new page of results.

  3. Call the /api/latest endpoint to view a list of the ten most recent searches.

  4. For Example:

    Basic Image Search->

    https://imapi.glitch.me/api/imagesearch/the%20office%20funny

    Image Search w/ Offset Query ->

    https://imapi.glitch.me/api/imagesearch/the%20office%20funny?offset=50

    Output Structure ->

    { "url": "someURL","snippet": "someHeadline", "thumbnail": "someURL", "context": "someURL" }

    Latest Searches ->

    https://imapi.glitch.me/api/latest
